# Uber-Dashboard
Uber Ride Analytics Dashboard | End-to-End Power BI Project
I designed and developed a comprehensive Power BI dashboard to analyze Uber ride operations across multiple vehicle types including Auto, Bike, Go Mini, Go Sedan, Premier Sedan, and Uber XL.

 Key KPIs Tracked
* Completed Bookings
* Lost / Cancelled Bookings
* Total Revenue
* Total Distance Covered
* Average Ride Distance
* Customer & Driver Ratings
  
Core Analysis & Insights
* Revenue analysis by vehicle type, payment method, customer, and location
* Monthly & quarterly trends for bookings, revenue, and distance
* Time-slot analysis (hour-wise & day-wise ride distribution)
* Pickup & drop location performance
* Customer segmentation: first-time, return, and regular riders
* Cancellation reasons analysis (customer-side & driver-side)

Power BI Features Used
* Advanced DAX measures & calculated columns
* Interactive slicers, filters, and bookmarks
* Drill-through & cross-filtering
* Custom visuals and clean UI/UX design
* Multi-page dashboard navigation
